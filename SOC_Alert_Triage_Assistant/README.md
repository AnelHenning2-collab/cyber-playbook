# SOC Alert Triage Assistant

## Overview
The SOC Alert Triage Assistant is designed to enhance the efficiency of security operations center (SOC) analysts by automating the triage of alerts. This tool leverages machine learning algorithms to prioritize alerts based on severity and potential impact.

## Features
- Automated alert triage
- Severity scoring system
- Integrates with existing SOC tools
- User-friendly dashboard

## Installation
1. Clone the repository.
2. Install dependencies using `pip install -r requirements.txt`.
3. Run the application with `python app.py`.

## Usage
- Input alert data through the dashboard.
- Review prioritized alerts and take action accordingly.
