LLM Fine‑Tuning Lab

Overview

The LLM Fine‑Tuning Lab provides a clean workflow for training, evaluating, and deploying custom language models for cybersecurity tasks.

Features

  • Dataset preprocessing and cleaning
  • LoRA and QLoRA fine‑tuning
  • Evaluation and metrics dashboard
  • Streamlit UI for testing outputs

Run the App

uvicorn api:app --reload --port 8002
streamlit run main.py
