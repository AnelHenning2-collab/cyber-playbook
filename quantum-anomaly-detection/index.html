Project Goal

Detect Rare, High-Impact Threats

Quantum kernel methods can capture subtle, high-dimensional patterns that classical models miss. This project explores quantum-inspired anomaly detection for SOC logs, network telemetry, and rare-event threat signals.

Simulated Impact: Improved detection of low-frequency, high-severity anomalies.

Architecture

Hybrid Quantum–Classical Workflow

The model uses quantum kernels to map data into a high-dimensional Hilbert space, enabling separation of subtle anomalies. 

# Pseudo-code for quantum anomaly detection
from qiskit_machine_learning.kernels import QuantumKernel
from sklearn.svm import OneClassSVM

quantum_kernel = QuantumKernel(feature_map=my_feature_map)
kernel_matrix = quantum_kernel.evaluate(X_train)

model = OneClassSVM(kernel="precomputed").fit(kernel_matrix)
