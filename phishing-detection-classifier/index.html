Phishing Detection Classifier

Overview

This phishing detection model classifies email subject and body text as phishing or legitimate using a lightweight NLP pipeline. It is designed for fast inference and easy integration into SOC workflows.

Features

  • Binary phishing classification
  • Fast inference with a compact transformer model
  • REST API backend (FastAPI)
  • Streamlit UI for testing email samples
  • Supports noisy, real‑world email text

Run the App

Start the backend and UI locally:

uvicorn api:app --reload --port 8001
streamlit run main.py
