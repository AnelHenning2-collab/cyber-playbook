Multimodal Security Assistant

Overview

The Multimodal Security Assistant processes text, images, and metadata to support SOC investigations with richer context and improved threat understanding.

Features

  • Image + text joint analysis
  • Embeddings-based similarity search
  • Metadata extraction and enrichment
  • Streamlit UI for uploading files

Run the App

uvicorn api:app --reload --port 8003
streamlit run main.py
